# 🧑‍💻 GitHub Profile API Card

A sleek React component that fetches and displays GitHub profile data using Recoil for state management. Built with modular components and responsive styling to create a dynamic user experience.

---

## 🚀 Features

- 🔍 Enter any GitHub username to fetch public profile details
- ⚛️ State managed via Recoil atom family
- 🎯 Displays avatar, bio, location, and repo stats
- 🧪 Loading and error states with fallback UI
- 🧩 Modular design: easily extendable with debounce, validation, animations, etc.

---

## 🧠 Concepts Covered
- useRecoilValueLoadable for async state handling
- atomFamily for dynamic state management
- Conditional rendering based on API response
- Modular component structure (GithubCard)


---

## 📦 Installation & Running

1. **Navigate to the frontend folder:**
   ```bash
   cd frontend
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the app locally:**
   ```bash
   npm run dev  # for Vite
   # or
   npm start     # if using CRA
   ```

4. **Open in Browser:**
   Navigate to [http://localhost:5173](http://localhost:5173) (or shown URL) in your browser.
